## 简介

`@univerjs/sheets-filter` 是 Univer Sheet 的筛选插件。

## 使用指南

你应该将此插件与 `@univerjs/sheets-filter-ui` 包一起使用。

### 安装

```shell
# 使用 npm
npm install @univerjs/sheets-filter

# 使用 pnpm
pnpm add @univerjs/sheets-filter
```
